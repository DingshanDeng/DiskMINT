# Surface Density Fitting Guide

*This section is a placeholder. Full content will be added after the DiskMINT-GARDEN grid paper is published.*

---

DiskMINT can be used to derive the disk surface density profile of a target by fitting observed dust continuum and C¹⁸O line profiles to a grid of models. The approach was developed and applied in [Deng et al. (2025), ApJ 995, 98](https://ui.adsabs.harvard.edu/abs/2025ApJ...995...98D); refer to that paper for the methodology and a worked example on IM Lup.

A step-by-step fitting guide — including how to set up the model grid, define observational constraints, and run the inference — will be documented here once the grid paper and associated DiskMINT inference tools are released (see Project 2 in the development roadmap).
