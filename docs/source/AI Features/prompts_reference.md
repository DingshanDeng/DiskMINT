# Prepared Prompts

A set of ready-to-use prompts for each DiskMINT-Nursery feature is maintained in the [`DiskMINT-Nursery` repository](https://github.com/DingshanDeng/DiskMINT-Nursery/tree/main/prompts).
Copy a prompt into your AI assistant (e.g. Claude Code) to get guided help for that task.

*Prompts are under development and will be added in an upcoming release.*

| Feature | Prompt file | What it does |
|---|---|---|
| Installation & Onboarding | [`install_prompts.md`](https://github.com/DingshanDeng/DiskMINT-Nursery/blob/main/prompts/install_prompts.md) | Check and set up your full DiskMINT environment |
| Runtime Assistant | [`assistant_prompts.md`](https://github.com/DingshanDeng/DiskMINT-Nursery/blob/main/prompts/assistant_prompts.md) | Get help running models and interpreting outputs |
| Support Escalation | [`escalation_prompts.md`](https://github.com/DingshanDeng/DiskMINT-Nursery/blob/main/prompts/escalation_prompts.md) | Diagnose errors and draft a support request |

---

For more on DiskMINT-Nursery and how the skill works, see the [DiskMINT-Nursery README](https://github.com/DingshanDeng/DiskMINT-Nursery).
